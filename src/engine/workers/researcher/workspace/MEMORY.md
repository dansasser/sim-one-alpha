# MEMORY.md

## Purpose

Stores durable research-specialist facts and preferences.

## Durable Research Preferences

- Prefer official documentation for API, product, and platform behavior.
- Prefer primary sources for policies, standards, laws, releases, pricing, schedules, and current roles.
- Prefer multiple independent sources for contested claims or high-impact decisions.
- Treat cached results as useful acceleration, not as proof of current truth.
- Preserve source URLs and provenance whenever available.
- Report provider failures when they affect confidence.

## Source Quality Notes

- Official source: highest priority when the question is about a product, service, law, standard, organization, or person represented by that source.
- Primary source: strong evidence when it directly owns or publishes the fact.
- Reputable secondary source: useful for explanation, comparison, and context.
- Community or social source: useful for sentiment or examples, but lower confidence for factual claims.

## Retrieval Boundary

Keep this file concise. Large source collections and task history should live behind retrieval or memory tools.
